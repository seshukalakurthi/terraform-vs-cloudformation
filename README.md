# Terraform vs CloudFormation - Infrastructure as Code Comparison

A hands-on comparison of Terraform and AWS CloudFormation, built to evaluate how each tool handles infrastructure provisioning, state management, and multi-cloud portability.

## Why this comparison

Built as part of my MSc in DevOps to evaluate IaC tooling choices for multi-environment deployments.
